📌 Sobre o projeto
MTClicker é um autoclicker altamente personalizável criado do zero em Java. Diferente de ferramentas genéricas, cada detalhe da interface foi desenhado manualmente em pixel art — desde os botões até os ícones — tornando o projeto único visualmente e funcionalmente.
Utiliza a biblioteca JNativeHook para capturar eventos globais de teclado e mouse, funcionando em qualquer janela do sistema operacional — ideal para jogos que exigem muitos cliques rápidos, como PvP no Minecraft ou qualquer título que se beneficie de automação de cliques.

✨ Funcionalidades

✅ Clique automático com intervalo de tempo configurável </p>
✅ Captura global de input via JNativeHook (funciona em qualquer janela) </p>
✅ Contador de cliques em tempo real </p>
✅ Interface gráfica personalizada com sprites em pixel art desenhados à mão </p>
✅ Ativação/desativação por atalho de teclado </p>

🚀 Como rodar o projeto</p>
Pré-requisitos</p>

Java JDK 11 ou superior instalado</p>
Maven ou qualquer IDE Java (IntelliJ, Eclipse, VS Code)</p>

Instalação</p>
bash# Clone o repositório</p>
git clone https://github.com/Mthxxx/MTclicker.git</p>

# Entre na pasta do projeto</p>
cd MTclicker</p>

# Abra na sua IDE favorita e rode o arquivo principal</p>
# ou compile via terminal:</p>
javac -cp . Main.java</p>
java Main</p>

⚠️ O JNativeHook pode precisar de permissões de acessibilidade no macOS. No Windows e Linux funciona direto.</p>


🎮 Como usar</p>

Execute o programa</p>
Configure o intervalo de cliques desejado</p>
Pressione o atalho de teclado para ativar/desativar</p>
O contador exibe os cliques realizados em tempo real</p>
